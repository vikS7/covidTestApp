## React Native covidTestApp (DEPRECATED)


App made to track COVID19 cases across India.

API's used :
- https://covid19api.io/api/v1  (for country data) (DEPRECATED)
- https://akashraj.tech/corona/api_india (for state data)
- http://newsapi.org/
